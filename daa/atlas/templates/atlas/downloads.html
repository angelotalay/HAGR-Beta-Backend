Downloads

The following file contains all of the physiological, psychological and molecular changes in the Digital Ageing Atlas as of 17/09/2014. It does not contain the pathological changes as the data can be retrieved in a more accessible format from the CDC.

The file is formated as a tab-separated text file and can be downloaded from here.

Note: Multiple entries in a column (e.g. tissues) are separated by a comma (,). The exception to this rule are references, which due to the presence of commas in the text, are separated using the pipe (|) character.

If you are interested in access to the full database please contact us for more information.
